# Tic-Tac-Toe in Lean 4

A simple Tic-Tac-Toe game implemented in Lean 4.

# Developer Guide

1. Install [Lean 4](https://lean-lang.org/install/manual/)

2. Navigate to the `tic-tac-toe-lean` directory using command line interface

3. Run the program with the following command:
```
lake exe tic-tac-toe-lean
```
The program should display the following message:
```
Welcome to Tic Tac Toe! Enter Q at anytime to quit.
.-----------.	.-----------.
| 1 | 2 | 3 |	|   |   |   |
|---+---+---|	|---+---+---|
| 4 | 5 | 6 |	|   |   |   |
|---+---+---|	|---+---+---|
| 7 | 8 | 9 |	|   |   |   |
'-----------'	'-----------'
Player X's turn. Please pick a number to place your X: 
```
